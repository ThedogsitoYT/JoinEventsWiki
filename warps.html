    Create Warps

  • To create a warp you must take into account the following commands /setwarp < name > - /warp < name > - /delwarp < name >

  • The main commands are: /setwarp < name > - /warp < name > - /warps - /delwarp < name > - /listwarp

  • Set the warp

    To create the warp you have to execute the command /setwarp < name > where it says to set the name, put whatever you want. That name will later be used for the /warp < name of the warp > command, which will use the name you set when setting the warp. warp and that name will also be in the config to easily recognize that warp. example:

  • Delete the warp

    To remove the warp you just have to run /delwarp < name of the warp > and it will be removed from the menu, command and config
